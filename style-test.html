📊 样式检测说明

如何使用组件库：

  1. 安装：npm install ty-ui-mobile-cockpit@1.0.5
  2. 导入组件：import { Cockpit, Button, Input, Card } from 'ty-ui-mobile-cockpit';
  3. 重要：导入样式：import 'ty-ui-mobile-cockpit/dist/index.css';

常见问题：

  • ❌ 忘记导入CSS文件
  • ❌ CSS文件路径错误
  • ❌ 样式被其他CSS覆盖
  • ❌ 组件类名不匹配
